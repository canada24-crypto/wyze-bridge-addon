# Wyze Bridge (Host Mode)

This add-on runs the official Wyze Bridge container with true host networking on Home Assistant OS.

## Features
- Host networking for low-latency streaming
- MediaMTX UI on port 8888
- Wyze API login
- Motion events
- Audio support
- ON_DEMAND streaming
